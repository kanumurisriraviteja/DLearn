Array      	-> same data type(Generic)          , index based, 		Fixed length  -> 1D,multi D,Jaggged
ArrarList  	-> Different data type(non Generic) , index based, 		Any length
List       	-> same data type(Generic)          , index based, 		Any length

IEnumerable -> same data type(Generic)          , index based, 		Any length, Can't Data after initialized.
IQueryable  -> same data type(Generic)          , index based, 		Any length, Can't Data after initialized,Holds good to process large data.
ICollection -> same data type(Generic)          , non index based,  Any length
HashSet     -> same data type(Generic)          , index based, 		Any length, Duplicates are not stored
HashTable   -> Different data type(non Generic) , key value based,  Any length
Dictionary	-> same data type(Generic)          , key value based,  Any length
Stack       -> Generic,Non Generic              , LIFO based,       Any length
Queue       -> Generic,Non Generic              , FIFO based,       Any length

LinkedList  -> single linked list,double linked list,circular linked list
Tree        -> Binary Tree,AVL Tree
